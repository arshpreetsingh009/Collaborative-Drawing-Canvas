This project is a real-time collaborative drawing app where multiple users can draw together on the same canvas.
Anything drawn by one user appears instantly for all other connected users.
The app supports:
Live drawing synchronization
Visible cursors for other users
Shared undo and redo across all clients
It is built using HTML Canvas, JavaScript, Node.js, and WebSockets to handle real-time communication.
How to Run the Project
Requirements
Node.js (v18 or higher)
npm
Steps
Open the project folder in terminal
Install dependencies:
npm install
Start the server:
npm start
Open the app in your browser or you can just access it using: 
http://localhost:3000
or if you dont want to run locally,
https://collaborative-drawing-canvas-as8t.onrender.com
To test collaboration, open the same URL in multiple tabs or different browsers and draw at the same time.
